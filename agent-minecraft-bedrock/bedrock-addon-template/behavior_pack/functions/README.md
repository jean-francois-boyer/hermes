# Fonctions Bedrock

Place ici les fichiers `.mcfunction` si tu ajoutes un vrai behavior pack.

Exemples futurs :

- `setup.mcfunction` : règles du monde, scoreboard, spawn
- `quest_1.mcfunction` : progression quête 1
- `rewards.mcfunction` : récompenses

Pour une première version, tu peux simplement copier les commandes générées par l’agent dans des command blocks Minecraft Bedrock.
