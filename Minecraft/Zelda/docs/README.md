# Zelda / Aventure Fantasy — Bedrock

## Fichier principal recommandé

Importer dans Minecraft Bedrock :

```text
../export/monde-aventure-fantasy.mcworld
```

## Fichiers Bedrock disponibles

- `monde-aventure-fantasy.mcworld` — monde complet avec packs attachés.
- `monde-aventure-fantasy-mcworld.zip` — copie ZIP si l'extension `.mcworld` est refusée.
- `mission-aventure-fantasy.mcaddon` — add-on seul.
- `mission-aventure-fantasy-behavior.mcpack` — behavior pack seul.
- `mission-aventure-fantasy-resources.mcpack` — resource pack seul.
- `heros-aventure-fantasy.mcpack` — skin pack du héros.

## Commandes en jeu

```mcfunction
/function start
/function quests
/function village-kokiri-original
/function temple-foret
/function mont-volcan
/function sanctuaire-lac
```
