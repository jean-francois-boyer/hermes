# Minecraft Bedrock — fichiers organisés

Ce dossier regroupe les projets Minecraft générés dans le workspace.

## Dossiers

- `Zelda/` — monde aventure fantasy original inspiré Zelda, importable dans Bedrock.
- `Nyon/` — monde réaliste de Nyon, Suisse, basé sur OpenStreetMap + relief réel.

Dans chaque projet :

- `export/` contient les fichiers à importer dans Minecraft Bedrock (`.mcworld`, `.mcaddon`, `.mcpack`, `.zip`).
- `source/` contient les fichiers générateurs, données ou fonctions source.
- `docs/` contient les guides d'import.
