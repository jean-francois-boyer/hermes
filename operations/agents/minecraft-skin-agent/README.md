# Agent Skins Minecraft

Profil Hermes: minecraft-skin-agent

Rôle: créer des skins Minecraft PNG 64x64, previews et packs .mcpack importables.

Script principal:
/opt/data/workspace/agent-minecraft-bedrock/skin_agent.py

Commande de lancement:
HERMES_HOME=/opt/data hermes --profile minecraft-skin-agent

Exemple:
HERMES_HOME=/opt/data hermes --profile minecraft-skin-agent chat -q "Crée un skin chevalier IA bleu et or compatible Bedrock"
