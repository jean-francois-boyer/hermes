# Agent Structures Minecraft

Profil Hermes: minecraft-structure-agent

Rôle: générer des structures Minecraft Bedrock sous forme de fichiers .mcfunction: spawn, maisons, marché, donjon, mobs, coffres.

Script principal:
/opt/data/workspace/agent-minecraft-bedrock/structure_agent.py

Commande de lancement:
HERMES_HOME=/opt/data hermes --profile minecraft-structure-agent

Exemple:
HERMES_HOME=/opt/data hermes --profile minecraft-structure-agent chat -q "Génère un village médiéval avec spawn, marché et donjon"
