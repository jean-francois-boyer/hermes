# Agent Monde Minecraft Bedrock

Profil Hermes: minecraft-world-generator-agent

Rôle: créer le concept complet d’un monde Minecraft Bedrock compatible Switch via Realms.

Script principal:
/opt/data/workspace/agent-minecraft-bedrock/minecraft_agent.py

Commande de lancement:
HERMES_HOME=/opt/data hermes --profile minecraft-world-generator-agent

Exemple:
HERMES_HOME=/opt/data hermes --profile minecraft-world-generator-agent chat -q "Crée un monde médiéval fantasy compatible Switch avec 5 quêtes et 5 PNJ"
