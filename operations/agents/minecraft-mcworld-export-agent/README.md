# Agent Export Monde .mcworld

Profil Hermes: minecraft-mcworld-export-agent

Rôle: préparer un export .mcworld à partir d’un vrai monde modèle Bedrock.

Script principal:
/opt/data/workspace/agent-minecraft-bedrock/mcworld_export_agent.py

Commande de lancement:
HERMES_HOME=/opt/data hermes --profile minecraft-mcworld-export-agent

Exemple:
HERMES_HOME=/opt/data hermes --profile minecraft-mcworld-export-agent chat -q "Explique comment exporter mon monde Bedrock en .mcworld compatible Realms"
