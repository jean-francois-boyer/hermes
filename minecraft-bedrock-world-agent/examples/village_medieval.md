# Exemple de demande à donner à l'agent

Crée un monde Minecraft Bedrock compatible Nintendo Switch via Realms.

Je veux un village médiéval fantasy pour une série YouTube.

Contraintes :
- compatible Bedrock
- jouable sur Switch via Realms
- pas besoin de mods obligatoires
- facile à construire progressivement
- commandes simples

Contenu souhaité :
- une belle zone de spawn
- un village avec 8 bâtiments
- 5 PNJ avec dialogues
- 5 quêtes
- 2 donjons
- une économie avec émeraudes
- un boss final
- un plan de construction épisode par épisode
