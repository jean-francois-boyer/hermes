# Premier monde généré : Le Royaume des Blocs Oubliés

## 1. Nom du monde
Le Royaume des Blocs Oubliés

## 2. Pitch
Un village médiéval abandonné reprend vie grâce au joueur, qui doit restaurer le royaume et ouvrir le portail de l'ancien donjon.

## 3. Zones principales

- Spawn : x 0, y 70, z 0
- Village central : x 80, z 0
- Mine abandonnée : x 180, z 90
- Forêt magique : x -120, z 80
- Donjon des Pierres Noires : x 250, z -150
- Château final : x 400, z 0

## 4. Zone de spawn
Construire une petite place en pierre avec :
- fontaine centrale
- panneau d'accueil
- coffre de départ
- chemin vers le village

Commande utile :
/give @p bread 16
/give @p wooden_sword 1
/title @p title Bienvenue dans le Royaume des Blocs Oubliés

## 5. PNJ

1. Maire Armand : donne la quête principale.
2. Lila la Forgeronne : vend armes et armures.
3. Milo le Marchand : échange nourriture contre émeraudes.
4. Sœur Eliane : soigne le joueur.
5. Garde Roland : ouvre l'accès au donjon.

## 6. Quêtes

1. Retrouver 10 pains pour nourrir le village.
2. Réparer le pont avec 32 blocs de bois.
3. Explorer la mine et rapporter 5 lingots de fer.
4. Battre les monstres de la forêt magique.
5. Trouver la clé du donjon final.

## 7. Économie

Monnaie : émeraudes.

Exemples de prix :
- pain x8 : 1 émeraude
- épée en fer : 8 émeraudes
- bouclier : 6 émeraudes
- potion de soin : 4 émeraudes

## 8. Commandes Bedrock simples

/spawnpoint @p 0 70 0
/tp @p 80 70 0
/give @p emerald 5
/effect @p regeneration 10 1
/summon zombie 250 70 -150

## 9. Checklist Switch / Realms

1. Créer ou importer le monde dans Minecraft Bedrock sur PC/mobile.
2. Tester les commandes.
3. Sauvegarder le monde.
4. Ouvrir Minecraft Realms.
5. Remplacer le monde du Realm par ce monde.
6. Se connecter à Minecraft sur Switch avec le même compte Microsoft/Xbox.
7. Rejoindre le Realm depuis l'onglet Amis/Realms.
